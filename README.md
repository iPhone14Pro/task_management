# 프로젝트 소개
난이도와 중요도에 따라 하루 태스크를 관리할 수 있도록 하는 웹 앱
- 프로젝트별, 목적별로 완수율, 주별 완수율 확인할 수 있음


## 사용 기술 스택
- 런타임: [Node.js](https://github.com/nodejs/node)
  - 오픈소스 라이브러리, 프레임워크: [npm](https://github.com/npm/npm)으로 다운받음
- 프론트엔드 프레임워크: [React](https://github.com/facebook/react)
  - 링크: [React Router](https://github.com/remix-run/react-router)

- 호윤
  - 페이지별 정보구조 설계
    - 정보구조: 페이지 연결 관계, 페이지별로 들어갈 정보들을 정의한 그림.

- 수민
  - 디자인 시안 보고 기능명세서 작성
  - 발표 자료 제작 및 발표

### 향후 프로젝트 진행 계획
- 1주차(11월 24일~30일) 동안에는 태스크를 추가하고 수정하고 삭제할 수 있는 기능을 완성

### 11월 23일 프로젝트 진행 일지

- 메인 페이지에서 공부시간 기능은 학생이 아니라 대학생, 직장인 타겟으로 한 앱이어서 삭제

- 프로젝트 선택 대화상자 디자인함
